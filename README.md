# radio
trysailのラジオを聞き逃さないように録音したい！ってことで作るradio録音

## やりたいこと
- 設定ファイルの作成と読み込み
- 番組表の取得，キーワード録音
- 複数番組の同時録音
- systemctlでの自動起動，再起動
- 出来たらクラウドストレージに保存（id, passは設定ファイル）
- 録音完了を何らかの方法で通知する（lineかなぁ）

---

・rec_radiko.sh
  $1=チャンネルコード
  $2=録音時間
  $3=保存先パス
  $4=ファイル名

・rec_agqr.sh
  $1=録音時間
  $2=ファイル名

保存先がSAVE_DIRに記述してあるのでそこを直せば使えます。

**start.shは関係ないです**
